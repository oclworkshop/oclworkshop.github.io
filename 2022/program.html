OCL Workshop

21st International Workshop on OCL and Textual Modeling, October 23-25 2022, Montreal, Canada

Workshop Program

8:30 - 10:00 Session 1: OCL & other textual constraint languages

  • Opening
  • Jordi Cabot, David Delgado and Lola Burgueño. Combining OCL and Natural Language: a Call for a Community Effort
  • Edward Willink. Challenges for Code Generated OCL Execution
  • Vanessa Tietz and Bjoern Annighoefer. A Formally Defined and Formally Provable EBNF-based Constraint Language for Use in Qualifiable Software

10:00 - 10:30 Coffee break

10:30 - 12:00 Session 2: Verification and validation

  • Frédéric Jouault, Maxime Méré, Matthias Brun, Théo Le Calvar, Matthias Pasquier and Ciprian Teodorov. From OCL-based Model Static Analysis to Quick Fixes
  • Kunxiang Jin and Kevin Lano. Design and Classification of Mutation Operators for OCL Specification
  • Mustafa Al Lail, Antonio Rosales, Hector Cardenas, Lars Hamann and Alfredo Perez. Transformation of TOCL Temporal Properties into OCL

12:00 - 13:30 Lunch break

13:30 - 15:00 Session 3: Extending OCL

  • Edward Willink. Support for OCL Libraries and Static Features
  • Kevin Lano, Shekoufeh Kolahdouz Rahimi and Kunxiang Jin. OCL libraries for software specification and representation
  • Lars Hamann, Martin Gogolla and Mustafa Al Lail. Categorization of Approaches to Reuse and Extend OCL

15:00 - 15:30 Coffee break

15:30 - 17:00 Session 4: Discussion
